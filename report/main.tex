\noindent
The 88M-parameter $\textsc{CNN}$ was fine-tuned on $\mathcal{D}_{\text{train}}$,
validated on $\mathcal{D}_{\text{val}}$, and tested on $\mathcal{D}_{\text{test}}$, where:\\

\[
\begin{gathered}
|\mathcal{D}_{\mathrm{train}}| = 448{,}000 \text{ diagrams}, \\[0.5em]
|\mathcal{D}_{\mathrm{val}}| = |\mathcal{D}_{\mathrm{test}}| = 56{,}000 \text{ diagrams}, \\[0.5em]
\mathcal{D} = \mathcal{D}_{\mathrm{train}} \cup \mathcal{D}_{\mathrm{val}} \cup \mathcal{D}_{\mathrm{test}}.
\end{gathered}
\]\\


\noindent
Each split in $\mathcal{D}$ contained an equal number of knots with $n \in N$ crossings where $N=\{20, 21, \dots ,35\}$. Each
split contained an equal number of non-trivial knots and unknots for all values of $n$.
