Extract the `train.zip`, `val.zip`, and `test.zip` data splits into the `SeeingTheUnknot/knot_data` directory to form:

```text
SeeingTheUnknot/
├─ knot_data/

│  ├─ val/
│  └─ test/
└─ ...
```
